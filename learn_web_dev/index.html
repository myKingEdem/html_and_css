Learn Full-Stack Web Development.

A guide to learning Ruby, Rails and related technologies systematically.

Learn the skills you need to become a full-stack web developer.

Stop getting frustrated with your lack of progress. This guide will teach you how to systematically learn Ruby, Rails, HTML & CSS, jQuery, JavaScript and RSpec, the technologies you need to know as a full-stack Ruby on Rails developer.

Training programs such as General Assembly, gSchool, DevBootCamp and The Starter League that teach web development cost anywhere between $8,000 and $36,000. Not everyone can quit their jobs and attend these programs. If you've ever wanted to learn web development, but don't have the time and resources to attend these programs, this book will guide you through learning everything you need to know to become a full-stack Ruby on Rails developer. It uses rapid skill acquisition principles to help you acquire the skills necessary to become a full-stack web developer.

  2. HTML & CSS.
  3. jQuery and JavaScript.
  4. Ruby on Rails.
  5. Testing with RSpec.
  6. Practical Advice for Developers.
  7. Applying Rapid Skill Acquisition Principles.

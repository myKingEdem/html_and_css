Training programs such as General Assembly, gSchool, DevBootCamp and The Starter League that teach web development cost anywhere between $8,000 and $36,000. Not everyone can quit their job and pay lots of money to attend these programs. If you've ever wanted to learn web development, but don't have the time and resources to attend these programs, this book will guide you through learning everything you need to know to become a web developer. It uses a self-paced approach, effective teaching and learning methods, and the best resources available to guide you through learning web development.
